# Prompt Blocks (coherencia + variedad)

Usar estos bloques para construir prompts consistentes. Mantener “look” premium y evitar ruido.

## Bloque A — Identidad (constante)

- Estilo: “premium futuristic tech, minimal, dark theme, high-end product visual”
- Materialidad: “matte black + subtle glass UI + thin neon accents”
- Iluminación: “controlled studio lighting, soft rim light, volumetric subtle haze”
- Calidad: “ultra clean, crisp edges, realistic reflections, high contrast, negative space”

## Bloque B — Protagonista (avatar)

- “Main subject: my personal brand avatar/character, centered hero composition”
- “Avatar must be the focal point, sharp, high detail, confident posture”
- Si el avatar es un archivo existente: “compose using the provided avatar image; do not redesign the character”

## Bloque C — Motivo firma (rotar 1)

- Grid: “technical grid overlay, thin lines, blueprint precision”
- HUD: “minimal HUD frame, corner marks, scanning line”
- Code: “abstract code pattern, terminal glow, readable-like but not cluttered”
- Dashboard: “minimal dashboard cards, charts, glassmorphism subtle”
- Flow: “automation flow nodes, connectors, clean diagram”

## Bloque D — Familia de fondo (variar 1)

- “dark gradient + subtle particles”
- “volumetric smoke + neon rim”
- “glass blur UI layers + depth”
- “blueprint background + wireframe”

## Bloque E — Tipografía (integrada)

- “bold sans headline, tight tracking, high legibility on mobile”
- “microcopy smaller, lighter weight, aligned to grid”
- “text integrated into composition, not floating randomly”

## Bloque F — Negativos (siempre)

- “no messy clutter, no low-res, no extra characters, no cartoonish, no cheap neon, no busy typography”
- “avoid watermark, avoid illegible text blocks, avoid oversaturated colors”

## Plantilla: Prompt final (composición)

1) A (identidad) + B (avatar)  
2) Elegir 1 de C (motivo) + 1 de D (fondo)  
3) Añadir layout (L1–L4) + E (texto) + F (negativos)
